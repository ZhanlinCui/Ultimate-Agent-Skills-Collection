# README Outline Guide

Recommended structure for bilingual release docs:

1. Title + tagline
2. Badges (license, runtime, platform)
3. Language switcher links
4. Overview / 项目简介 (1 short paragraph)
5. Contents / 内容 (list key deliverables or submodules)
6. Key Capabilities / 关键能力 (bullets)
7. System Components / 系统组件 (bullets)
8. Quick Start / 快速上手 (short command block)
9. Configuration / 配置 (list config files)
10. Project Structure / 目录结构 (tree snippet)
11. License / 许可证
12. Credits / 鸣谢 (optional)

When missing data, add TODO placeholders and ask the user to fill them.
