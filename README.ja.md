## 📖 Skills とは？

Skills は、AI エージェントが動的に読み込んで特定のタスクのパフォーマンスを向上させる指示、スクリプト、リソースのフォルダです。Claude（または他の AI エージェント）に特定のタスクを再現可能な方法で完了する方法を教えます - ドキュメントの作成、データの分析、ワークフローの自動化、アプリケーションの構築など。

## 🌟 なぜこのコレクション？

これはインターネット上で **最も包括的に厳選された高品質** Agent Skills コレクションです：

- ✅ **40以上のプレミアム Skills** - クリエイティブ、技術、エンタープライズワークフローをカバー
- ✅ **本番環境対応** - これらの Skills は実際のアプリケーションを動かしています
- ✅ **充実したドキュメント** - 各 Skill には詳細な説明と例が含まれています
- ✅ **カテゴリ別に整理** - 必要なものを簡単に見つけられます

## 🚀 使用方法

### Claude.ai で

1. プロジェクト設定 → Skills に移動
2. skill フォルダまたは個別の `SKILL.md` ファイルをアップロード
3. Claude が各 skill をいつ使用するかを自動検出

### Claude Code で

```bash
# プラグインマーケットプレイスとして登録
/plugin marketplace add your-skills-repo

# または直接インストール
/plugin install skill-name@your-marketplace
```

### Claude API で

[Skills API クイックスタート](https://docs.claude.com/en/api/skills-guide) に従って Claude API 経由で skills を使用。

## 📁 Skill 構造

各 skill はこの標準構造に従います：

```
skill-name/
├── SKILL.md          # メイン指示（必須）
├── scripts/          # ヘルパースクリプト（オプション）
├── templates/        # テンプレートファイル（オプション）
├── references/       # ドキュメント参照（オプション）
└── examples/         # 使用例（オプション）
```

`SKILL.md` ファイルの内容：

```markdown
---
name: skill-name
description: この skill をいつ使用するか、何をするか
---

# Skill 名

[Claude が従う指示]
```
